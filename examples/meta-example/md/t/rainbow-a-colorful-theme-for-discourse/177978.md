This theme is inspired by Rainbows – a symbol of beauty, peace and happiness. ![:rainbow:](https://emoji.discourse-cdn.com/twitter/rainbow.png?v=15)

Since [Discourse is a system of Rainbows](https://blog.discourse.org/2013/02/the-discourse-logo-a-system-of-rainbows/), I wanted to create a theme with pastel rainbow colors. 🙂
